## Introduction
### Brief introduction to Vivado tool  

After completing this lab you will be able to:

- Create a new Vivado project
- Understand Vivado design flow
- Add source, constraint and simulation files
- Synthesize, implement and generate bitstreams
- Program device 
- Source control your project

*Note: This lab is not a comprehensive user guide for Vivado IDE. The lab had been deliberately simplified to help new users understand the system without interfacing unnecessary complexities.*
